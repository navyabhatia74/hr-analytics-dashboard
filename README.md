# HR ANALYTICS DASHBOARD
## Overview
This Hr Analytics Dashboard was built using Microsoft Excel to analyze employee attrition,overtime trends,workfoce demographics,and job satisfaction insights through interactive visualizations.
##KEY INSIGHTS
-Employee attrition analysis
-Overtime impact on workforce retention
-Department-wise attrition trends
-Job satisfaction distribution
-Workfoce demographic overview
## DASHBOARD FEATURES
-KPI Crads
-Interactive Slicers
-Donut Charts
-Bar Charts
Workforce analytics visualizations 
## Tools Used
-Microsoft Excel 
-Pivot Tabels
-Pivot Charts
-Data Visualization
